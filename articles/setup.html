Setup guide for preparing to run a service

Overview

This checklist covers all the standard setup procedures needed to run a service. After completing the Startup checklist, follow these steps to complete all the necessary imports and connections.

Step 1

Load the appropriate show on the soundboard. See Loading a Show for more help

Step 2

Turn on the camera input in ProPresenter

  • Open the Media Bin
  • Scroll to the bottom of the media playlists
  • Click the "Video Inputs" playlist
  • Trigger Input 1

Step 3

Establish the MIDI connection for Lighykey control via ProPresenter

Skip this step if you have a lighting operator
  • Launch Audio MIDI Setup on the Lightkey MacBook Pro
  • In Sessions and Directories, click ProPresenter Mac
  • Click Connect

Step 4

Aircast the ProPresenter output to the Lobby TV

Make sure the Lobby TV is turned on!
  • Start screen mirroring from the Mac's display menu
  • Toggle the screen color in Screens > Configure Screens

Step 5

In System Settings, set the audio output to Scarlett 2i2

Step 6

Check that all screens, camera inputs, lights, MIDI connections, and audio inputs/outputs work as expected

Step 7

Add or import all your presentations into the ProPresenter service playlist

Step 8

Paste the Restream stream key into ProPresenter's capture settings

  • In Safari, open Restream and login if required
  • Find the event scheduled for that service
    • If no event is scheduled, let your team lead know
  • Copy the stream key from the event details
  • Paste the stream key into ProPresenter Live > Capture Settings > Steam Key

Step 9

Choose appropriate motion backgrounds for each song and assign lyric macros as needed

Step 10

Test all slides, videos, countdowns, macros, backgrounds, etc.
